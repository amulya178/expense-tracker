# expense-tracker
A command-line app to log, view, and analyze your personal expenses. No internet, no server, no extra installs — just Python's built-in sqlite3 module.

Features:
  Add expenses with amount, category, description, and date (auto-filled)
  View all expenses, sorted newest first
  See a spending summary by category, with percentages and a text bar chart
  Delete an expense by ID
  Data persists between runs in a local expenses.db file
